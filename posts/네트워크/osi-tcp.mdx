# OSI 7개층이란 ?
OSI 7 계층(OSI 7 Layer)는 네트워크 프로토콜이 통신하는 구조를 7개의 계층으로 
분리하여 각 계층간 상호 작동하는 방식을 정해 놓은 것 이다. 
<p align="center">
  <img src="/img/network/osi.jpg" width="500"/>
</p>

## 물리 계층 (Physical Layer)
* 데이터를 전기적인 신호로 변환해서 주고 받는 기능을 수행한다.
* 통신 단위는 비트를 사용하며 이 비트는 0과 1로 표현된다.
* 장비에는 통신 케이블, 리피터, 허브 등이 있다.

💨 <u>**물리 계층은 데이터를 전기적인 신호로 변환하고 물리적으로 전송하는 역할**</u>

## 데이터 링크 계층(DataLink Layer)
* 물리 계층에서 전송된 비트들을 논리적인 데이터 프레임으로 정리하고 관리하여 상위 계층에 전달한다.
* 프레임에 Mac 주소를 부여하고 에러 검출, 재전송, 흐름제어를 진행한다.
* 데이터 링크 계층에서 사용되는 주요 장비에는 브리지(Bridge), 스위치(Switch), 이더넷 등이 있다.

💨 <u>**데이터 링크 계층은 네트워크에서 데이터의 프레임 단위 관리와 오류 검출, 흐름 제어를 담당.**</u>

## 네트워크 계층(Network Layer)
* 데이터를 안전하게 전송하는 역할을 한다.
* 라우터(Router)를 통해 경로를 선택하고 IP 주소를 이용하여 패킷(Packet)을 목적지로 전달
* 데이터의 전송 단위는 **패킷**

💨 <u>**네트워크 계층(Network Layer)은 데이터를 안전하게 라우팅하여 목적지로 전달하는 역할을 수행.**</u>

## 전송 계층(Transport Layer)
* 신뢰할 수 있는 데이터 전송이 목적이다.
* 프로토콜로 TCP와 UDP를 사용
* 전송 계층에서는 포트 번호를 사용하여 서버의 서비스를 식별

💨 <u>**전송 계층(Transport Layer)은 데이터의 신뢰성과 흐름을 관리하며, 출발지에서 목적지로 데이터를 안전하게 전송하는 역할.**</u>

## 세션 계층(Session Layer)
* 통신 세션을 설정, 유지, 종료하며 데이터 통신을 조절.
* 통신의 시작과 끝을 담당하며 동기화와 체크 포인트 설정을 수행하여 데이터의 신뢰성을 유지.

💨 <u>**세션 계층은 통신 세션 설정과 관리를 통해 데이터 통신의 시작과 끝을 조절하며, 동기화와 체크 포인트 설정으로 데이터 신뢰성을 유지.**</u>

## 표현 계층(Presentation Layer)
* 전송하는 데이터의 표현방식을 결정(압축,암호화 및 보안)
* 데이터의 인코딩, 디코딩, 암호화, 복호화등의 기능을 수행

💨 <u>**표현 계층은 데이터의 표현 방식을 결정하며, 데이터의 인코딩, 디코딩, 암호화, 복호화 등 다양한 기능을 담당하여 데이터의 형식과 보안을 관리.**</u>

## 응용 계층(Application Layer)
* 최종 사용자 및 응용 프로그램에 서비스를 제공하는 역할.
* HTTP, SMTP, POP3, FTP등의 프로토콜이 속한다.

💨 <u>**응용 계층은 사용자와 응용 프로그램 간의 통신을 지원하며, 표준 프로토콜을 사용하여 데이터 전송과 서비스를 관리**</u>

# TCP/IP 4계층
네트워크 관리 및 통신에서 주로 사용되는 것은 TCP/IP 모델이다.

<p align="center">
  <img src="/img/etc/tcpip.jpg" width="500"/>
</p>

## 네트워크 액세스 계층(Network Access Layer)
* OSI 7계층의 물리계층과 데이터 링크 계층에 해당
* MAC 주소를 사용하여 장치를 식별
* 데이터 패킷을 전기신호로 변환하여 선로를 통하여 전달할 수 있게 준비 

## 인터넷 계층(Internet Layer)
* OSI 7계층의 네트워크 계층에 해당
* 라우팅은 IP 주소를 기반으로 이루어지며, 최적의 경로를 선택하여 데이터를 전달
* IP (Internet Protocol) 주소를 사용하여 호스트와 네트워크를 식별
* 라우팅 및 IP 주소 할당을 통해 데이터 패킷의 안정적인 전달하며, 네트워크 간의 통신을 지원

## 전송 계층(Transport Layer)
* OSI 7계층의 전송 계층에 해당
* TCP/UDP 사용
* 데이터의 신뢰성과 속도를 조절하기 위한 중요한 역할

## 응용 계층(Application Layer)
* OSI 7계층의 세션 계층, 표현 계층, 응용 계층에 해당한다.
* 다양한 서비스 및 응용 프로그램을 제공(이메일, 파일 전송, 동영상 스트리밍, 게임)
* 프로토콜에는 HTTP, SMTP, POP3, FTP, DNS 등을 가진다.
