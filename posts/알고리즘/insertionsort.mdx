# 삽입 정렬이란 ?
작은 숫자부터 차례대로 삽입하여 정렬하는 방법이다. 이름 그대로 배열 안에 원소를 '삽입'하여 정렬하는 방식으로 작동한다

## 시간 복잡도
평균적으로 O(n2)

## 작동 원리
* 배열의 두 번째 원소부터 시작하여 첫 번째 원소까지 비교해가며 삽입할 위치를 찾는다
* 삽입할 위치를 찾았다면, 해당 위치에 원소를 '삽입'하고, 원래 위치에 있던 원소들을 한 칸씩 뒤로 옮긴다.
* 이 과정을 반복하여 모든 원소가 정렬될 때까지 진행한다.

[5,3,4,1,2] 라는 배열이 있으면<br></br> 
초기 인덱스 5,3을 비교해서 정렬 [3,5],4,1,2<br></br>
세번쨰요소인 4를 정렬된 부분 [3,5]와 비교해서 [3,4,5], 1,2<br></br>
네번재 요소인 1을 정렬된 부분 [3,4,5]와 비교해서 [1,3,4,5], 2<br></br>
마지막 요소를 정렬된 부분 [1,3,4,5]와 비교해서 [1,2,3,4,5]

## 코드
```js
function insertionSort(arr) {
  for (let i = 1; i < arr.length; i++) {
    const key = arr[i];
    let j = i - 1;
    while (j >= 0 && arr[j] > key) {
      arr[j + 1] = arr[j];
      j--;
    }
    arr[j + 1] = key;
  }
}
```
