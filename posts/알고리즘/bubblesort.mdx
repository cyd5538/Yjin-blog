# 버블 정렬이란 ?
버블 정렬은 인접한 두 원소를 비교하여 큰 값을 오른쪽으로 이동시키면서 배열을 정렬하는 방법이다.
버블 정렬의 이름은 정렬 중 큰 값이 "거품처럼" 배열의 끝까지 올라가는 형상을 연상시키기 때문에 붙여졌다.
## 시간 복잡도
**O(n^2)**
## 기본 동작 원리
배열의 모든 값을 다음 값과 비교하고 큰 수를 다음으로 보낸다<br></br>
[29, 10, 14, 30, 37, 14, 18] 시작<br></br>

[10, 29, 14, 30, 37, 14, 18] (인덱스 0, 1 비교) 29와 10 비교하여 스왑<br></br>
[10, 14, 29, 30, 37, 14, 18] (인덱스 1, 2 비교) 29와 14 비교하여 스왑<br></br>
[10, 14, 29, 30, 37, 14, 18] (인덱스 2, 3 비교) 29와 30 비교하여 변경 없음<br></br>
[10, 14, 29, 30, 37, 14, 18] (인덱스 3, 4 비교) 30와 37 비교하여 변경 없음<br></br>
[10, 14, 29, 30, 14, 37, 18] (인덱스 4, 5 비교) 37과 14 비교하여 스왑<br></br>
[10, 14, 29, 30, 14, 18, 37] (인덱스 5, 6 비교) 37과 18 비교하여 스왑<br></br>

최대값 37이 마지막 자리에 정렬<br></br>
다음 반복에서는 마지막 원소를 제외하고 정렬 진행<br></br>
이 과정을 계속 반복한다.
## 코드
```js
function bubbleSort(arr) {
  const n = arr.length;

  for (let i = 0; i < n - 1; i++) {
    for (let j = 0; j < n - 1 - i; j++) {
      if (arr[j] > arr[j + 1]) {
        [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]]; // 스왑
      }
    }
  }

  return arr;
}
```
